# 修仙Roguelike - 游戏设计文档

## 游戏定位

横版动作Roguelike游戏，以修仙世界观为背景，单机运行，在网页浏览器中游玩。主要操作方式为电脑键盘+鼠标，同时兼容手机触屏操作。游戏采用像素风格，第一版全部使用色块占位，不依赖任何图片资源。

玩家扮演一名散修剑客，通过不断进入秘境历练，在随机生成的地图中战斗、成长，收集灵石与机缘词条，逐步提升境界，最终得证大道。

---

## 核心循环

```
局外大厅（修炼/升级）→ 进入秘境（随机地图）→ 横版战斗 → 
过关抽机缘词条 → 死亡或通关 → 带灵石回局外 → 循环
```

整个游戏围绕"局外养成 → 局内冒险 → 局外再养成"的循环展开：

1. **局外大厅**：玩家在修炼洞府中消耗灵石提升境界、解锁新词条池、升级基础属性（攻击力、血量、灵力上限）。
2. **进入秘境**：每次进入秘境时，地图房间随机生成，包括战斗房、精英房、商人房、机缘房和Boss房。
3. **横版战斗**：在2D横版场景中进行实时动作战斗，使用普攻、闪避、技能和大招应对不同类型的敌人。
4. **奖励获取**：击败Boss后抽选机缘词条（三选一），词条在本局内生效，叠加构筑独特流派。
5. **结算回归**：死亡或通关后，保留局内获得的灵石，回到局外大厅继续养成。

## 境界系统（局外永久成长）

境界是局外永久成长的核心，玩家消耗灵石突破境界，每个境界解锁新的词条池和被动加成。

| 境界 | 解锁条件 | 解锁词条 | 被动加成 |
|------|----------|----------|----------|
| 炼气期 | 初始 | 基础词条池（10张） | 无 |
| 筑基期 | 消耗500灵石 | 新增5张词条 | 攻击力+10% |
| 金丹期 | 消耗2000灵石 | 新增5张词条 | 最大血量+15% |
| 元婴期 | 消耗8000灵石 | 新增5张词条 | 灵力恢复+20% |
| 化神期 | 消耗30000灵石 | 新增5张词条 | 全属性+10% |

**灵石消耗策略**：玩家需要在"提升基础属性"和"突破境界解锁新词条"之间做出取舍，每一轮的选择都影响后续的游戏体验。

## 角色（第一版）

### 散修剑客

一名以剑入道的散修，剑法凌厉，身法灵动，兼具爆发与生存能力。

**普攻：剑气斩**
- 三段连击，伤害递增：第一段80%攻击力 → 第二段100%攻击力 → 第三段130%攻击力
- 每次连击之间有0.3秒的间隔窗口，按下攻击键可触发下一段
- 第三段附带小幅击退效果

**闪避：身法步**
- 向前方瞬移一小段距离，期间无敌帧持续0.4秒
- 冷却时间：2秒
- 可取消普攻后摇

**技能：御剑术**
- 向前方发射一道穿透剑气，对直线上的所有敌人造成180%攻击力伤害
- 冷却时间：4秒
- 射程覆盖2/3屏幕宽度

**大招：剑域**
- 以自身为圆心释放剑阵领域，对范围内所有敌人造成300%攻击力伤害
- 消耗灵力：50点（灵力上限初始100点）
- 冷却时间：15秒
- 施放期间获得50%伤害减免

**基础属性**：
- 攻击力：100
- 最大血量：500
- 灵力上限：100
- 灵力恢复：每秒2点
- 移动速度：200像素/秒
- 暴击率：5%
- 暴击伤害：150%

## 词条系统（机缘）

### 机制说明

每间Boss房过关后，触发机缘抽卡：从当前境界解锁的词条池中随机抽取3张，玩家选择1张在本局内生效。词条效果可叠加，同名词条不重复出现。机缘房也可以免费获得一张词条，由系统随机赋予。

### 第一版词条（共10张）

| 编号 | 名称 | 效果 |
|------|------|------|
| 1 | 【烈阳剑气】 | 普攻附加灼烧，每秒造成5%攻击力伤害，持续3秒 |
| 2 | 【鬼步身法】 | 闪避后无敌帧延长至0.8秒 |
| 3 | 【铜皮炼体】 | 受击有25%概率免疫本次伤害 |
| 4 | 【三叠剑意】 | 第三段普攻伤害×2.5 |
| 5 | 【吸星大法】 | 击杀敌人回复5%最大血量 |
| 6 | 【雷法附灵】 | 御剑术改为雷属性，命中后使敌人眩晕0.5秒 |
| 7 | 【金钟罩】 | 血量低于30%时获得护盾（护盾值为最大血量的40%），每局触发一次 |
| 8 | 【疾风步】 | 移动速度+30% |
| 9 | 【灵力汇聚】 | 大招灵力消耗-40% |
| 10 | 【剑心通明】 | 暴击率+20%，暴击伤害×1.8 |

## 秘境地图（第一版）

### 地图结构

每次秘境由9个房间组成：8个随机房间 + 1个Boss房（固定为第9间）。

房间之间的连接采用线性+分支结构：玩家从起点出发，每通过一个房间后有两个可选路线（左/右），最终汇聚到Boss房。

### 房间类型

| 类型 | 出现概率 | 内容 |
|------|----------|------|
| 战斗房 | 50% | 3-5只随机小怪（山鬼/毒蛊/邪修），全灭后开门 |
| 精英房 | 15% | 1只精英怪（小怪强化版），必掉灵石（50-100） |
| 商人房 | 15% | 用灵石向商人购买丹药恢复血量（小还丹30灵石恢复20%血量，大还丹80灵石恢复50%血量） |
| 机缘房 | 10% | 免费获得一张随机词条 |
| Boss房 | - | 固定第9间，妖王熊罴，通关后触发机缘抽卡（三选一） |

### 商人丹药

| 丹药 | 价格 | 效果 |
|------|------|------|
| 小还丹 | 30灵石 | 恢复20%最大血量 |
| 大还丹 | 80灵石 | 恢复50%最大血量 |

## 敌人（第一版）

### 小怪

**山鬼**
- 类型：近战
- 行为：锁定玩家位置后冲刺攻击，冲刺速度较快
- 攻击力：20
- 血量：80
- 掉落：5-15灵石

**毒蛊**
- 类型：远程
- 行为：与玩家保持距离，每2秒吐出一颗毒球（直线飞行，命中造成中毒持续伤害）
- 攻击力：15（毒球直伤）+ 每秒5点中毒伤害持续3秒
- 血量：50
- 掉落：5-15灵石

**邪修**
- 类型：近战+格挡
- 行为：接近玩家进行近战攻击，被攻击时有30%概率举剑格挡（减免70%伤害），格挡后反击
- 攻击力：25
- 血量：100
- 掉落：10-20灵石

### Boss

**妖王熊罴**

一只修炼成精的巨大熊妖，分为三个阶段：

**第一阶段（血量100%-66%）**
- 技能1：熊掌拍地 → 前方扇形范围冲击波，造成40伤害
- 技能2：冲刺撞击 → 直线冲刺，碰到墙壁后短暂眩晕1秒
- 行为模式：拍地 → 冲刺 → 普攻两下 → 循环

**第二阶段（血量66%-33%）**
- 保留第一阶段的全部技能，新增：
- 技能3：妖气爆发 → 以自身为圆心释放绿色毒雾，持续5秒，玩家进入范围每秒受到15伤害并减速30%
- 行为模式加快20%

**第三阶段（血量33%-0%）**
- 保留前两阶段的全部技能，新增：
- 技能4：狂暴 → 攻击力+50%，移动速度+30%，但受到伤害+25%
- 行为模式变得不可预测（随机选择技能）

**Boss属性**：
- 总血量：1500（每阶段500）
- 攻击力：40（狂暴后60）
- 通关掉落：100-200灵石 + 机缘抽卡

## 局外资源

### 灵石

**获取途径**：
- 击杀敌人掉落（小怪5-20，精英50-100，Boss 100-200）
- 通关额外奖励（按剩余血量百分比加成）

**消耗途径**：
- 突破境界（炼气→筑基500，筑基→金丹2000，金丹→元婴8000，元婴→化神30000）
- 提升基础属性：
  - 攻击力+5：消耗100灵石
  - 最大血量+20：消耗100灵石
  - 灵力上限+10：消耗100灵石

### 仙玉

**获取途径**：
- 首次通关每个关卡奖励仙玉
- 境界突破奖励仙玉
- 每日登录奖励（后续实现）
- 特殊成就奖励（后续实现）

**消耗途径**：
- 机缘阁抽卡（丹药池30仙玉、法宝池50仙玉、天材地宝池80仙玉、功法池60仙玉）

### 属性升级

玩家在局外大厅可以自由分配灵石进行属性升级，每次升级对应属性的提升幅度固定，价格递增（每次购买后该属性价格+20灵石）。

## 局内流程

1. **进入秘境**：在局外大厅点击"进入秘境"按钮
2. **地图导航**：在地图界面选择房间路线（每层二选一）
3. **战斗**：进入横版战斗场景，击败所有敌人
4. **商人/机缘**：进入特殊房间进行交易或领取词条
5. **Boss战**：第9间挑战妖王熊罴
6. **通关结算**：击败Boss后抽选词条，可选择继续深入（第二版）或返回大厅
7. **死亡结算**：血量归零时游戏结束，保留灵石返回大厅

## 第一版开发优先级

1. **项目搭建**：Vite + Phaser 3 项目初始化，空场景跳转
2. **玩家控制**：普攻三段连击、闪避、技能、大招
3. **敌人AI**：山鬼、毒蛊、邪修的基础行为
4. **战斗系统**：伤害计算、状态效果、击杀掉落
5. **地图系统**：MapScene路线选择，房间随机生成
6. **词条系统**：机缘抽卡UI，词条效果实现
7. **局外系统**：HallScene境界突破、属性升级
8. **Boss战**：妖王熊罴三阶段AI
9. **完整循环**：局外→局内→结算→局外流程打通
10. **手机适配**：触屏虚拟按键、UI缩放
